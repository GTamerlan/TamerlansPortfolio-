My 3D Project

This project is an interactive 3D world built using Three.js. It showcases a Martian-like environment and a rover that responds to user interactions. You can explore the terrain and see the rover in action with sound effects.

The project integrates advanced features such as:

  • Custom gradient skybox
  • Interactive environment rotation
  • Ambient sound effects for the rover
  • Responsive design for a seamless user experience

Feel free to interact with the world and explore!
